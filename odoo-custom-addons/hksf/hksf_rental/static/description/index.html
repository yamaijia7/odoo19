HKSF Rental

Rental quotation module for Odoo 19 — rebuilt from the Odoo 11 odoo_sale_quote module chain.

Key Features

  • Sale type toggle: Rental vs Sales
  • Charge type: Monthly or Weekly
  • Per-line rental duration (months) auto-calculated from start/end date
  • Rental amount = unit price × months × qty (excl. tax)
  • Weight & volume tracking per line and as order totals
  • Rental / Sale line sections on PDF report
  • Custom header logo, company stamp and chop-and-sign on PDF
  • Salesperson signature image on PDF footer
  • Subject line & subject description on quotation
  • Grand total custom label
